Development Showcase

Over the years I've developed hundreds of automations and SaaS solutions within the Enterprise and SMB cybersecurity industries. Below you will find a showcase of the projects I am most proud of and actively working on.

BlockerBoy

BlockerBoy is an adaptive custom NIDS (network intrusion detection system) & malicious network traffic blocker created from scratch using python. I developed it during an active attack, real-time, during the attack, in order to mitigate malicious DD0S activities.

AutoAudit

AutoAudit is an automated security audit tool designed for SMB and enterprise organizations. It is built with Python and incorporates popular security tools such as Nmap, OpenVAS, and ZAP to perform vulnerability assessments, risk assessments, and compliance audits. AutoAudit generates comprehensive HTML reports and can be scheduled to run at regular intervals. The tool can also send email notifications with the latest audit reports. I developed AutoAudit to save an organization time and money as well as to ease the workload of security engineers in the SMB and Enterprise markets.

ICSZeroTrustAudit

I developed ICSZeroTrustAudit to perform an automated security audit on a target Industrial Control System (ICS). It includes multiple tests and checks, such as vulnerability scanning, network traffic analysis, credential checking, firewall and Intrusion Detection System (IDS) testing, and ICS-specific protocol testing (Modbus, DNP3, and IEC 60870-5-104). Additionally, it performs a compliance check based on Zero Trust principles.

IOCM8

IOCM8 scans a windows system for known IOC (indicators of compromise) outlined within the script. IOCM8 defines a list of known IOCs and then iterates through each IOC to check if it is present on the system. If the IOC is present, the script prints information about the IOC, such as its properties, network connections, or DNS queries, provides remediation instructions, and helpful resources from reputable security resources. I developed IOCM8 to help engineers keep their machines clean as well as hold the ability to remain on the cutting edge of emerging threats.
